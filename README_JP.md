## アクセスカウンターについて

本サイトは、2000年代初頭の個人サイトの雰囲気や構造をできる限り忠実に再現することを目的とした作品です。  
その一環として、当時多く使われていた外部アクセスカウンター **RAYS COUNTER** を使用しています。

### GitHub Pages でカウンターが表示されない理由

本サイトは **GitHub Pages（HTTPS）** 上で公開されていますが、  
**RAYS COUNTER** は主に **HTTP 環境** を前提として作られた、非常に古いサービスです。

HTTPS ページから HTTPS に完全対応していない外部リソースを読み込もうとすると、  
現在のブラウザではセキュリティ上の理由により通信がブロックまたは中断されます
（Mixed Content / 接続リセット等）。

そのため、

- ローカル環境（`file://` や `http://localhost`）では表示される場合がある
- GitHub Pages（HTTPS）ではカウンターが正常に表示されない

という挙動になります。

これはサイトの実装ミスや GitHub Pages の設定によるものではなく、  
**元のカウンターサービス自体が HTTPS 環境に対応していないこと**が原因です。

### それでもカウンターを残している理由

本プロジェクトは、最新技術や利便性を重視するものではなく、  
**当時の個人サイト文化を再現・体験すること**を目的としています。

アクセスカウンターは、その時代を象徴する重要な要素のひとつであるため、  
現在の環境では動作に制限があっても、  
**雰囲気再現・ノスタルジー要素としてあえて残しています。**

### 注意事項

- 本サイトは個人の学習・再現目的の非商用プロジェクトです
- 実際のユーザーデータ収集は行っていません
- カウンターは実際のアクセス数を示すものではありません

ご理解いただき、ありがとうございます。
